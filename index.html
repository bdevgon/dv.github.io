About the Visualizaton

About
The visualiazation is about city of New Orleans in Louisiana. The pages attempts to show insights about the city.It is a template , more visualizations can be plotted as per availability of data. It tries to enagage the user by showing several insights about the city. The attempt is to provide extensive data driven insights about the city for assisting users in making informed decisions about the city. Similar visualizations can be developed for several cities across the world and user can view the concerns and opportunities around them. More infromation to come soon.

Structure
The project story follows a drill down story structure. The user can interact with the visualizations on individual pages and look for insights about the city.

Scenes
Each scene in the visualization show cases information about the city of New Orleans. More insights can be plotted and the users of this visualization can take informed decisions about living in New Orleans. The scenes contain compelling visualizations rendering from data obtained from data.gov.

Annotations
Annotations are presented in terms of tool tips on both area chart and bar charts

More About the Visualizaton !

Parameters
The buttons on the bar chart slide section allows users to switch between parameters and factors and re-render the view. The same svg container is re render based on different parametes

Triggers
Each time user pushes a button on bar chart visualization an event "trigger" is fired which calls appropriated function in javascript to render the bars. Also it removes the previous visualization already present in the SVG container Tooltips attempt to show demonstrate same behaviour.
